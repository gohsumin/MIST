# Setup
1. Run `npm install`
2. Download [twgl.min.js](https://github.com/greggman/twgl.js/tree/master/dist/4.x), and put it in the public folder

# Testing server
Run `npm start`

# Building
Run `npm build`

# Problems with webgl
Move `src/mistnogl.js` to `src/mist.js` (but, don't commit this change)
